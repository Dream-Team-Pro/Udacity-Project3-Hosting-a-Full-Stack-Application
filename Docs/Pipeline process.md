# Pipeline Process

### From the root of the project:

```
- npm run frontend:install              To install frontend dependencies
- npm run frontend:start                To start frontend dependencies
- npm run frontend:build                To build frontend dependencies
- npm run frontend:test         
- npm run frontend:e2e          
- npm run frontend:lint         
- npm run frontend:deploy               To deploy the project to S3 using ./udagram-frontend/bin/deploy.sh deploy script
- npm run backend:install               To install backend dependencies           
- npm run backend:build                 To build backend dependencies        
- npm run backend:start                 To start backend dependencies   
- npm run backend:deploy                To deploy the project to EB using ./udagram-api/bin/deploy.sh deploy script  
- npm run deploy            


```
